# Endless Horizon
## Pitch
Endless Horizon est un jeu se déroulant dans un monde ouvert et fictif fortement
inspiré de l’âge d’or de la piraterie du 17ème siècle. On y incarne un capitaine pirate
en quête de richesses et d’aventures. Le jeu offre une liberté de déplacement totale
que ce soit à pied sur les nombreuses îles du monde ou en mer, aux commandes
des différents types de navires qui suivent une physique réaliste.
Le monde est un océan infini rempli d’îles de tailles variées et est entièrement généré procéduralement grâce à divers algorithmes aléatoires, ce
qui permet de rendre chaque expérience unique.
Le joueur pourra explorer de vastes étendues maritimes et combattre les navires qu’il croisera.

## Fonctionnalités clés
-  Monde infini procédural
-  Navigation physique
-  Déplacements sur terre et sur mer
-  Batailles navales

## Installation
Il suffit de télécharger le fichier zip dans "release", de l'extraire et de lancer EndlessHorizon.exe

## Contrôles
### A pieds
Les déplacements se font avec les touches **ZQSD** et la caméra se déplace par les mouvements de la souris. Pour piloter un navire, il faut appuyer sur la touche **E** en étant face au gouvernail de ce dernier. 
### En mer
Pour diriger un navire, il faut utiliser les touches **ZQSD**. La touche **V** permet d'ouvrir ou de fermer les voiles et un **clic gauche** permet de tirer avec les canons dans la direction pointée par la caméra.

## Code
Le jeu est programmé en C# avec le moteur Unity. Les scripts des systèmes implémentés se trouvent dans le dossier [Scripts](Assets/Scripts).
